# D&D Tool (Placeholder Name)
This Dungeon & Dragons Tool is a set of useful programs that will be useful for D&D sessions 
especially for Dungeon Master but also for normal player. It helps keeping track of summoned 
characters without relying on too many different apps.


## Prerequisites

Python 3.9 or newer
Internet access for initial Database creation

## Installation & Usage

```
pip install -r requirements.txt
python sync_spells.py
python sync_items.py
python sync_monsters.py
python gui/app.py
```

## Features
- Dice Roller 
- Management System for characters
- Spell database
- Monster database
- Database for created characters
- GUI
